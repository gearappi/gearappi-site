Что такое GEARAPPI?

Это самый простой хостинг статики, который вы могли себе представить, с поддежкой пользовательских серверных функций для работы с базой данных, уведомлениями и т.д.
Простая папка на вашем компьютере - это доступ к вашему хостингу, куда вы cможете загружать файлы и они будут синхронизироваться с сервером.

Новый подход к WEB

Вам не надо знать DevOPS, работайте локально и все файлы будут автоматически загружаться на сервер.

Круглосуточная техническая поддержка

Мы знаем, что для стабильной работы сайта, всегда нужно быть наготове, для этого мы круглосуточно отслеживаем работу серверов.

Фокусировка на одержании побед

Наша основная цель дать возможность творцам легко воплощать свои идеи, создавать не отвлекаясь, дать возможность сохранять фокусировку

Мобильное приложение

Отслеживайте статистику вашего сайта с телефона, приложение работает на Android и iOS.

Back-end как сервис

Создавайте пользовательские функции для работы с уведомлениями и базой данных на графическом программированнии

Удобный интерфейс

Мы ценим Ваше время и отдаем огромное внимание на то чтобы наш интерфейс был максимально интуитивным.

Технологии

  • Хостинг статики (html, css, js)
  • Работа с уведомлениями (Email, SMS)
  • Базы данных SQL (NoSQL)
  • Back-end функции
  • Аналитика сайта
  • Бесплатные домены третьего уровня c SSL
  • Подключение пользовательского домена с SSL

Как Начать

api.gearappi

on github api.gearappi

Для работы с api, вам потребуется склонировать репозиторий


git clone https://github.com/gearappi/api.gearappi.git

Далее перейдите в эту директорию и установите все пакеты


$ npm install

Запустите сервер


# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
